iOS Shortcut

Build a Shortcut named Save to Potpuri. In Shortcut Details, enable Show in Share Sheet, then set Receive to URLs, Safari webpages, and text. If it does not appear in an app's share sheet, add Any while testing.

  1. Add URL and set it to the endpoint below.
  2. Add Get Details of Shortcut Input for Name. Use that result as the title form field.
  3. Add Get Details of Shortcut Input for URL. Use that result as the url form field.
  4. Add Text containing Shortcut Input. Use that result as the text form field.
  5. Add Get Contents of URL. Set method to POST, request body to Form, and add the fields below.

Shortcut endpoint:
{{.BaseURL}}/api/shortcut

Form fields:
token = {{.NewToken}}
title = Name from Shortcut Input
url = URL from Shortcut Input
text = Shortcut Input

Use Get Contents of URL, not Share or Share with Apps. Those actions hand content to another app; Potpuri needs a POST request to the endpoint above. Run JavaScript on Webpage is optional and Safari-only; it can extract selected text or page metadata, but it still needs to pass the result into the same POST request.
