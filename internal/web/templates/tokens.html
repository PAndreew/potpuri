iOS Shortcut — one shortcut for everything

Just two actions. Encode Base64 converts anything iOS gives you — photos become image bytes, URLs and text become readable strings. The server detects which is which.

  1. In Shortcut Details enable Show in Share Sheet and set Receive to Any.
  2. Add Encode Base64 (search "Base64"). Input: Shortcut Input, mode: Encode.
  3. Add Get Contents of URL. Method: POST, body: JSON, URL: {{.BaseURL}}/api/shortcut.
  4. Add two JSON fields:
    • token → {{.NewToken}}
    • image → Base64 Encoded (output of step 2)

Share a URL → saved as a link. Share text → saved as a note. Share a photo → stored as an encrypted attachment. No branching or type-checking needed.
