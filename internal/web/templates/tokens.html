iOS Shortcut — one shortcut for everything

This single shortcut handles URLs, text, and photos from any share sheet. The server detects image bytes automatically, so always sending base64 is safe — text and URLs are silently ignored as non-image data.

  1. In Shortcut Details enable Show in Share Sheet and set Receive to Any.
  2. Add Get Details of Shortcut Input → Name. Save result as ItemTitle.
  3. Add Get Details of Shortcut Input → URL. Save result as ItemURL.
  4. Add Encode Base64 (search "Base64"). Input: Shortcut Input, mode: Encode. Save result as Encoded.
  5. Add Get Contents of URL. Method: POST, body: JSON, URL: {{.BaseURL}}/api/shortcut.
  6. Add these JSON fields:
    • token → {{.NewToken}}
    • title → ItemTitle
    • url → ItemURL
    • image → Encoded

No branching needed. Sharing a URL saves a link. Sharing text saves a note. Sharing a photo stores it as an encrypted attachment.
