Docs

Potpuri saves links, notes, photos, and files in a minimal private library.

Saving items

Use Add to Potpuri to save a note, link, photo, or file. Titles are optional; Potpuri will choose one from the URL, filename, or first line of text when possible.

Searching

The search bar filters visible entries as you type. Submitting the form also runs the server-side encrypted token search.

API tokens, bookmarklet, and iOS Shortcut

Create an API token from API Tokens. The same token can be used for the bookmarklet and the iOS Shortcut.

Export

Your account page includes JSON and bookmarks HTML exports.
